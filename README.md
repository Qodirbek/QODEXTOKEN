# QODEX Web App

Bu loyiha, foydalanuvchilarga tanga yig'ish va referal tizimi yordamida bonuslar olish imkonini beradi. Loyihada tanga yig'ish, energiya va o'yinlarni boshqarish uchun tugmalar mavjud.

## Asosiy xususiyatlar:
- **Home**: Bosh sahifa.
- **Earn**: Referal tizimi orqali tanga olish.
- **Upgrade**: Bosishni kuchaytirish va energiyani oshirish.
- **Wallet**: Foydalanuvchi hamyonini ulash.

## Texnologiyalar:
- **Backend**: Flask
- **Frontend**: HTML, CSS, JavaScript
- **Statik fayllar**: Rasmlar va stil (CSS)
